# 海外免费上线流程

推荐平台：Render

原因：
- 可以运行本项目的 Node/Express 后端
- 会自动赠送免费二级域名，例如 `https://sairuisi-official-site.onrender.com`
- 不需要 ICP 备案
- 支持 HTTPS

## 重要限制

Render 免费 Web Service 的本地文件是临时存储。服务重启、重新部署或休眠唤醒后，后台本地保存的 `leads.json` 可能丢失。

因此免费方案下，请务必配置邮箱 SMTP，让官网表单提交后立即同步到 `1760772194@qq.com`。

## 上线前准备

1. 注册或登录 GitHub
2. 把本项目推送到一个 GitHub 仓库
3. 注册或登录 Render
4. 使用 Render Blueprint 部署本仓库根目录的 `render.yaml`

## Render 需要填写的环境变量

`render.yaml` 已经预置大部分配置。部署时需要手动填写以下密钥：

```env
ADMIN_TOKEN=请改成你自己的后台口令
SMTP_USER=1760772194@qq.com
SMTP_PASS=QQ邮箱SMTP授权码
SMTP_FROM=1760772194@qq.com
```

已预设：

```env
LEAD_EMAIL_TO=1760772194@qq.com
SMTP_HOST=smtp.qq.com
SMTP_PORT=465
SMTP_SECURE=true
```

## QQ 邮箱授权码获取

1. 打开 QQ 邮箱网页版
2. 进入 设置 -> 账号
3. 找到 POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV 服务
4. 开启 SMTP 服务
5. 生成授权码
6. 将授权码填入 Render 的 `SMTP_PASS`

注意：`SMTP_PASS` 不是 QQ 登录密码。

## 上线后访问

官网：

```text
https://你的服务名.onrender.com/
```

后台：

```text
https://你的服务名.onrender.com/admin
```

后台口令就是 Render 环境变量里的 `ADMIN_TOKEN`。

## 健康检查

```text
https://你的服务名.onrender.com/api/health
```

返回 `{ "ok": true }` 即表示服务正常。
