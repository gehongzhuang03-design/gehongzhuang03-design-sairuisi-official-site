# 赛锐锶竞赛内容工作台

手机入口：`https://siris.onrender.com/studio`

## 安装到手机

- iPhone：用 Safari 打开入口，点底部“分享”，选择“添加到主屏幕”。
- 安卓：用 Chrome 打开入口，点右上角菜单，选择“安装应用”或“添加到主屏幕”。

首次打开时输入官网后台管理口令。口令会保存在手机本地，文字和生图 API 密钥只保存在服务端。

## 每天的使用流程

1. 每天 08:00 自动读取竞赛来源并筛选新信息，也可在首页点“立即采集最新信息”。
2. 打开候选选题，先查看原始来源，确认比赛仍在报名或征集中。
3. 点“生成今日推文”，系统会生成公众号长文、小红书短文、公众号横版封面和小红书方图。
4. 在手机上直接改标题和正文，离开输入框时自动保存。
5. 公众号点“去公众号发布”，复制文案后打开微信；小红书点“一键分享发布”，通过手机系统分享面板带上正文和配图。

## 发布边界

- 公众号官方接口直发需要公众号 AppID、AppSecret 和相应发布权限，当前版本先使用可靠的复制/打开发布助手。
- 小红书没有开放发布权限时，使用系统分享面板是更稳妥的手机方案，不采用模拟登录或绕过平台风控的脚本。
- 推文会主动标出待核验信息。报名时间、参赛对象、费用和赛制必须以原始通知为准。
